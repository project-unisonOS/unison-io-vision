# Security

This service is a vision I/O gateway for UnisonOS and is intended for local or tightly controlled deployments.

For overall security and privacy guidance, see:

- https://project-unisonos.github.io/reference/security/

If you believe you have found a security issue in unison-io-vision, please follow the responsible disclosure guidance in the UnisonOS security docs or open a private security advisory on GitHub rather than filing a public issue.
